# 🎂 生日派对互动网页模板

一个精美的宝宝生日派对庆祝网页模板合集，包含一周岁和两周岁的庆典页面。

---

## 📁 项目结构

```
BirthdayParty/
├── one-year.html    # 一周岁派对页面
├── two-year.html    # 两周岁派对页面
└── README.md        # 项目说明文档
```

---

## ✨ 功能特性

### 🎈 通用功能
- **照片上传与裁剪** - 点击照片区域可上传图片，支持裁剪成正方形
- **彩屑飞舞动画** - 持续飘落的多彩纸屑特效
- **动态背景** - 闪烁星光和唯美装饰
- **精美贴纸系统** - 多彩祝福语贴纸随机分布
- **图标点缀** - 气球、礼物、蛋糕等可爱图标装饰
- **响应式设计** - 适配各种屏幕尺寸

### 🎂 one-year.html (一周岁派对)
- 🔴 **红色主题** - 喜庆温暖的氛围
- 👑 **皇冠装饰** - 彰显小主角的可爱风采
- 🎊 **彩旗横幅** - "HAPPY 1ST" 字样
- 💝 **温馨祝福语** - 聪明伶俐、茁壮成长等

### 🎀 two-year.html (两周岁派对)
- 🟣 **紫色糖果主题** - 甜美梦幻的氛围
- 🪄 **魔法棒装饰** - 充满童趣的魔法元素
- 🎈 **彩虹气球** - 漂浮的彩色气球动画
- 📺 **全屏模式** - 点击按钮可进入全屏观看
- 💖 **糖果色系** - 粉、黄、蓝、橙等糖果色调

## 🚀 使用方法

### 方式一：直接打开
双击 `one-year.html` 或 `two-year.html` 文件，在浏览器中打开即可预览。

### 方式二：本地服务器
```bash
# 使用 Python 启动简易服务器
python -m http.server 8000

# 然后访问
http://localhost:8000/one-year.html
http://localhost:8000/two-year.html
```

### 方式三：修改祝福语
用文本编辑器打开 HTML 文件，搜索修改以下内容：
- 祝福语贴纸文字
- 底部标语
- 主标题文字

## 🎨 自定义修改

### 修改祝福语
```html
<div class="wish-tag tag-tape" style="...">🌟 修改这里</div>
```

### 修改主题色
```css
:root {
    --bg-red: #e63946;        /* 修改背景红色 */
    --yellow: #ffbe0b;        /* 修改黄色 */
    /* ... 其他颜色 */
}
```

### 修改照片框尺寸
```css
.photo-container {
    width: 420px;   /* 修改宽度 */
    height: 420px;  /* 修改高度 */
}
```
